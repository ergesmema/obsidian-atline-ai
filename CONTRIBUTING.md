# Contributing to AtLine AI

Thanks for your interest in contributing! This plugin is open to community input.

## Ways to Contribute

### Suggest Features or Improvements
- Open a [GitHub Issue](https://github.com/ergesmema/obsidian-atline-ai/issues) with the "enhancement" label
- Join [GitHub Discussions](https://github.com/ergesmema/obsidian-atline-ai/discussions) for open-ended ideas
- Vote on existing suggestions to help prioritize the roadmap

### Report Bugs
1. Check if the issue already exists
2. Open a new issue with:
   - Obsidian version
   - Plugin version
   - Steps to reproduce
   - Expected vs actual behavior
   - Error messages (from Developer Console: Cmd/Ctrl+Shift+I)

### Submit Code
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly in Obsidian
5. Submit a pull request

### Code Guidelines
- Keep changes focused and minimal
- Follow existing code style
- Test with multiple AI providers if applicable
- Update README if adding user-facing features

## Development Setup

1. Clone the repo into your vault's `.obsidian/plugins/` folder
2. Run `npm install` (if applicable)
3. Enable the plugin in Obsidian
4. Use Developer Console (Cmd/Ctrl+Shift+I) for debugging

## Questions?

Open a [Discussion](https://github.com/ergesmema/obsidian-atline-ai/discussions) or reach out via Issues.
